# Codex Implementation Context

This file is generated by `ticket-to-task --agent codex`.

Run:

```bash
npx tsx src/cli/cli.ts ticket-to-task <issueNumber> --agent codex
```

The generated version includes the GitHub issue URL, relevant files, roadmap context, acceptance criteria, and a Codex-ready implementation prompt.

## Usage in the Autonomous Loop

When `aicoder` processes an issue it calls `ticket-to-task` internally and passes the output to the coding agent. The coding agent should:

1. Implement the task described in the prompt
2. Run any relevant tests
3. Print `FIN` to stdout when complete — this signals `aicoder` to stop the agent and push the branch

If the agent exits without printing `FIN`, `aicoder` will skip the push and try again on the next poll cycle.
